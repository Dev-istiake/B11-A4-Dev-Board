Question-1: What are the different ways to select an element in the DOM?

There are 5 different ways to select an Html Element in DOM. These are: 1. getElementById, 2.getElementsByClassName, 3.getElementsByTagName, 4.querySelector, 5.querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHtml elements sets all Html content such as "Tag, class, id, text Content" within an element. While innerText and textContent sets only text within a html element, but innerText and textContent are not same. Difference between innerText and textContent: innerText sets only visible text within an element but avoid hidden text while textContent sets all text content including hidden text

Question-3: What is event delegation in the DOM?

Event delegation is a technique in javascript that allows to handel events more efficiently with the advantage of event bubbling. Event bubbling is a process of event handling. When an event occurs on an htmlElement, first of all it runs the event handler on that element then it runs on its parent, then all the way up to the root. Through event delegation, after adding an event listener to an HTML element, we can select its parent and also select all child elements.
