Question-1: What are the different ways to select an element in the DOM?

There are tow different ways to select Html Element in DOM. These are: 1. getElementById, 2.getElementByClassName, 3.getElementByTagName, 4.querySelector, 5.querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHtml element sets all Html content such as "Tag, class, id, text Content" within an element. While innerText and textContent sets only text within a html element, but innerText and textContent are not same. Difference between innerText and textContent: innerText sets only visible text within an element but avoid hidden text while textContent sets all text content including hidden text
